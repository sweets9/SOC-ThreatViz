# Screenshots Directory

This directory should contain the following screenshots for the README:

1. **dashboard.png** - Main dashboard showing the 3D globe with threat arcs, real-time feed, and statistics panels
2. **attack-details.png** - Detailed view of a specific attack showing source, target, service, and status information
3. **city-attacks.png** - View showing all attacks originating from or targeting a specific city with geographic pivot options

## Adding Screenshots

To add screenshots:

1. Take screenshots of the application
2. Save them with the exact filenames listed above
3. Place them in this directory
4. Commit and push to GitHub

The README will automatically display these images once they are added to the repository.
